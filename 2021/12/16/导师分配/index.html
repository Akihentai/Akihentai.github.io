Python编程实现学生随机分配导师

Python读取<待分配学生模板buu.xlsx>，<待分配导师模板buu.xlsx>中的内容，按照要求实现学生与导师之间的随机分配。随机分配的结果写入到<随机分配结果模板buu.xlsx>
注意：
1）<待分配学生模板buu.xlsx>，<待分配导师模板buu.xlsx>这两个文件中的行数并不固定。
2）一名学生对应一名导师，一名导师对应多名学生。
3）模板
附件1：待分配学生模板buu.xlsx
附件2：待分配导师模板buu.xlsx
附件3：随机分配结果模板buu.xlsx

解答：

import xlrd
import xlutils.copy
import random
from random import shuffle
from xlutils import copy
import xlwt
stu = xlrd.open_workbook(“1.xlsx”)
stu_sheet = stu.sheet_by_index(0)
stu_id = stu_sheet.col_values(0)
stu_name = stu_sheet.col_values(1)
stu_sex = stu_sheet.col_values(2)
del(stu_id[0])
del(stu_sex[0])
del (stu_name[0])
shuffle(stu_id)
tea = xlrd.open_workbook(‘2.xlsx’)
tea_sheet = tea.sheet_by_index(0)
tea_name = tea_sheet.col_values(0)
tea_member = tea_sheet.col_values(1)
del(tea_member[0])
del(tea_name[0])
rbook = xlrd.open_workbook(‘3.xlsx’)
wbook = xlutils.copy.copy(rbook)
w_sheet = wbook.get_sheet(0)
x=0
def find(a):
for i in range(len(stu_id)):
if int(stu_id[i]) == a:
return i

for i in range(len(stu_id)):
w_sheet.write(i+1,1,stu_id[i])
w_sheet.write(i+1,0,stu_name[find(int(stu_id[i]))])
w_sheet.write(i+1,2,stu_sex[find(int(stu_id[i]))])

for i in range(len(tea_name)):
for j in range(int(tea_member[i])):
w_sheet.write(x+1,3,tea_name[i])
x=x+1
wbook.save(‘03.xlsx’)
